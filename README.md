# GlowBalance ✨

**Split Smart. Glow Together.**

A beautiful, fun, and modern expense splitter & balance visualizer. Perfect for trips with friends, shared rent in Chattogram, family expenses, office lunches, or any group that wants to keep things fair without the headache.

### Why GlowBalance?
- Bright neon design that actually feels rewarding to use
- Smart minimal settlements (fewer transactions)
- Real-time balances with glowing visuals
- Beautiful animated charts (pie + bar)
- Multiple groups support
- Built with pure HTML, CSS (Tailwind), JavaScript + Node.js backend
- Default currency: ৳ BDT (Bangladeshi Taka)

### Features
- Create multiple groups
- Add members with custom colors
- Add expenses with flexible splitting (Equal / Unequal / Custom shares)
- Live balance calculation with color-coded cards
- Vibrant neon pie & bar charts powered by Chart.js
- Smart settlement suggestions
- Confetti animation when you settle debts
- Search, filter, edit & delete expenses
- Export summary as CSV
- Fully responsive & mobile-friendly
- Dark mode with electric neon accents

### Tech Stack
- **Frontend**: HTML5, Tailwind CSS, Vanilla JavaScript, Chart.js
- **Backend**: Node.js + Express.js + MongoDB (Mongoose)
- **Storage**: LocalStorage (for quick demo) + MongoDB
